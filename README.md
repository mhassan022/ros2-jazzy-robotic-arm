# ROS2 Jazzy Robotic Arm

This is my first robotics project and my entry into the robotics world.

The project includes a robotic arm built using ROS2 Jazzy with URDF modeling, simulation, and control.

## Features
- 4 DOF robotic arm
- URDF model
- Launch files
- Meshes and textures
- ROS2 package structure

## Package
- Adel_Arm
